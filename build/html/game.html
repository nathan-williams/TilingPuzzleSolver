Game Module¶

This is a custom made library to facilitate the graphics, solving, and general gameplay of the Tiling Puzzle Game.

Game Class¶

Manager for graphics and board/tile state.

clearBoard()[source]¶

Resets board and despawns all tiles.

drawBoard()[source]¶

Draws the board.

drawBoardAndTiles(tileBucket=None)[source]¶

Draws both the board and the tiles have been placed on the board.

Parameters:tileBucket (TileBucket) – used to determine if and where tiles should be drawn on the board
drawSolutions()[source]¶

Draws solutions that have been found and added to the solutions list.

drawTileOnBoard(tile)[source]¶

Draws a tile that has been both spawned and placed on the board based on its tile.row and tile.col.

Parameters:tile (Tile) – the tile to be drawn
drawTileOnCanvas(tile)[source]¶

Draws a tile that has been spawned, but not placed on the board, based on the tile.x and tile.y.

Parameters:tile (Tile) – the tile to be drawn
initializeSettings(settings)[source]¶

Initializes graphics constants and variables. If the settings parameter is empty, then default values are used.

load(fileKey)[source]¶

Loads the currently selected puzzle.

Parameters:filekey (str) – refers to the file containing the puzzle to be loaded
parseInputFile(filekey)[source]¶

Interprets the board configurations in the text file referred to by the fileKey. Each body of text is taken to be a tile where the largest tile is the board.

Parameters:filekey (str) – refers to the file containing the puzzle to be read and loaded
update(tileInfo=None)[source]¶

Redraws the board, tiles, and tile list.

Parameters:tileInfo (dict) – if provided, used to determine tile states; otherwise, self.tileBucket is used
